To run the website code, we used visual studio as our editor and github as our version control.
We always updated our versions before deploying to our web host to test new implemented features and code.
Preferebly using visual studio and a type of web hosting such as IIS or apache open the code as a website and debug starting from index.php.
We used our hosting service to test out new code.

The mobile system was developed in Eclipse with the Android SDK. 
To run the code, download the entire Traffic folder and open it in Eclipse with the Android SDK installed.
Select Window > AVD Manager from the Window dropdown menu.
Create a virtual device to test the code.
Select Run > Run As > Android Application from the Run dropdown menu to launch the application on the virtual device.
